# Ikigai Statement Template

---

## Ikigai Statement Formula

```
ฉันคือ [คุณสมบัติเด่น]
ที่ช่วย [กลุ่มคน]
ให้ [คุณค่าที่ให้]
ผ่าน [วิธีการ/สิ่งที่ทำ]
```

---

## วิธีเขียน

1. **คุณสมบัติเด่น** — คุณเป็นคนยังไง? (5 คำจับ essence)
   - เช่น: คนที่ตั้งคำถามเก่ง, คนที่อธิบายของยากให้เข้าใจง่าย, คนที่มี empathy สูง
   
2. **กลุ่มคน** — คุณอยากช่วยใคร? (เฉพาะเจาะจง)
   - เช่น: มนุษย์เงินเดือนไทย, นักเรียนที่เกลียดเลข, ฟรีแลนซ์มือใหม่
   
3. **คุณค่าที่ให้** — พวกเขาได้อะไรจากคุณ?
   - เช่น: ตัดสินใจได้ชัดขึ้น, มีสมาธิ longer, ประหยัดเวลา

4. **วิธีการ** — คุณทำยังไง?
   - เช่น: การวิเคราะห์ 5 มิติ, การสอนแบบกันเอง, การทำระบบ

## คำถามตรวจสอบ (หลังเขียนเสร็จ)

- [ ] อ่านแล้วรู้สึก **ตื่นเต้น** หรือ **โล่งใจ**?
- [ ] มันเป็น **จริง** สำหรับฉัน? (ไม่ใช่สิ่งที่ฉันอยากเป็น แต่สิ่งที่ฉันเป็น)
- [ ] วงกลมทั้ง 4 ข้อ สะท้อนอยู่ใน statement?
- [ ] ถ้าให้เพื่อนสนิทอ่าน เขาจะบอกว่า "นี่แหละนาย"?
