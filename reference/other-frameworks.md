# Frameworks อื่นๆ ที่ช่วยในการค้นหา Ikigai

---

## 1. Wheel of Life (วงล้อชีวิต)

### วิธีใช้
วาดวงกลมแบ่งเป็น 8 ส่วน แล้วให้คะแนนแต่ละด้าน 1-10:

```
        การงาน
          |
   การเงิน -+- สุขภาพ
          |
      ครอบครัว
```

| ด้าน | คำถาม | คะแนน (1-10) |
|------|-------|-------------|
| การงาน | งานปัจจุบันตอบโจทย์ชีวิตมั้ย? | |
| การเงิน | การเงินคล่องตัวมั้ย? | |
| สุขภาพ | ร่างกายแข็งแรง, จิตใจสบาย? | |
| ครอบครัว | ความสัมพันธ์ในครอบครัวเป็นไง? | |
| ความรัก | ความสัมพันธ์โรแมนติก? | |
| เพื่อน | มีสังคม / เพื่อนที่ดี? | |
| พัฒนาตัวเอง | ได้เรียนรู้ของใหม่? | |
| สนุก | มีเวลาทำสิ่งที่ชอบ? | |

**จุดประสงค์:** หาว่าชีวิตขาดสมดุลด้านไหน — เพื่อให้ Ikigai ไม่ได้ทำให้ด้านอื่นพัง

## 2. StrengthsFinder (CliftonStrengths)

### หลักการ
แทนที่จะแก้จุดอ่อน — **ลงทุนกับจุดแข็ง**

### 34 Strengths (แบ่งเป็น 4 กลุ่ม)

| กลุ่ม | ตัวอย่าง |
|------|---------|
| **Executing** | Achiever, Arranger, Discipline, Focus |
| **Influencing** | Command, Communication, Competition, Woo |
| **Relationship Building** | Empathy, Developer, Includer, Positivity |
| **Strategic Thinking** | Analytical, Futuristic, Ideation, Learner |

### วิธีใช้
1. หาจุดแข็ง 5 อันดับแรกของคุณ (ทำแบบทดสอบ)
2. ดูว่า Ikigai ของคุณ ใช้จุดแข็งเหล่านี้มั้ย?
3. ถ้าไม่ — มีทางปรับ Ikigai หรือใช้จุดแข็งอื่นแทน?

## 3. Holland Code (RIASEC)

### หลักการ
คนแบ่งเป็น 6 ประเภทอาชีพ:

| Code | ประเภท | ลักษณะ | อาชีพตัวอย่าง |
|------|--------|--------|-------------|
| **R** | Realistic | ชอบทำงานกับมือ, นอกอาคาร | ช่าง, เกษตรกร, เชฟ |
| **I** | Investigative | ชอบคิด, วิเคราะห์, วิจัย | นักวิทยาศาสตร์, นักวิเคราะห์ |
| **A** | Artistic | ชอบสร้างสรรค์, อิสระ | ศิลปิน, นักเขียน, นักออกแบบ |
| **S** | Social | ชอบช่วยคน, สอน, ดูแล | ครู, พยาบาล, นักจิตวิทยา |
| **E** | Enterprising | ชอบเป็นผู้นำ, ต่อรอง, ขาย | ผู้จัดการ, พนักงานขาย, ผู้ประกอบการ |
| **C** | Conventional | ชอบระบบ, ระเบียบ, ตัวเลข | นักบัญชี, เจ้าหน้าที่ธุรการ |

### วิธีใช้
1. เลือก 3 ตัวที่ตรงกับคุณมากที่สุด (เช่น SIA = ชอบสอน+วิเคราะห์+สร้างสรรค์)
2. ใช้เป็นตัวกรอง Ikigai — Ikigai ของคุณสอดคล้องกับ Holland Code หรือไม่?

## 4. 5 Whys (Root Cause Analysis)

### วิธีใช้
ถาม "ทำไม" 5 ครั้งเพื่อหาเหตุผลลึกของสิ่งที่คุณอยากทำ:

**ตัวอย่าง:**
- ทำไมอยากเป็นครู? → "เพราะอยากช่วยคน"
- ทำไมอยากช่วยคน? → "เพราะตอนเด็กไม่มีใครช่วยเรา"
- ทำไมตอนเด็กไม่มีใครช่วยเราถึงสำคัญ? → "เพราะเราต้องฝืนเรียนในสิ่งที่เราไม่ชอบ"
- ทำไมการฝืนเรียนถึงแย่? → "เพราะเราเสียเวลา 12 ปีกับสิ่งที่ไม่ได้ใช้"
- ทำไมการเสียเวลาถึงแย่? → "เพราะเราอยากให้เด็กรุ่นหลังไม่เสียเวลาแบบเรา"

→ **Ikigai แท้จริง:** ช่วยเด็กให้ไม่เสียเวลากับระบบการศึกษาที่ไม่ตอบโจทย์

## 5. Regret Minimization Framework (Jeff Bezos)

### หลักการ
เวลาต้องตัดสินใจใหญ่ — ถามตัวเอง:

> "เมื่ออายุ 80 ผมจะเสียดายมั้ยถ้าไม่ได้ลองทำสิ่งนี้?"

### วิธีใช้กับ Ikigai
1. ถ้า Ikigai ของคุณต้องการการเปลี่ยนแปลงใหญ่
2. ถามตัวเอง: "อีก 30 ปี จะเสียดายมั้ยที่ไม่ยอมลอง?"
3. ถ้าคำตอบคือ "เสียดาย" — ไปเลย

## 6. OODA Loop (John Boyd)

### หลักการ
Observe → Orient → Decide → Act

### วิธีใช้กับ Ikigai
| Step | ในบริบท Ikigai |
|------|---------------|
| **Observe** | สังเกตตัวเอง: ตอบคำถาม 4 หมวด |
| **Orient** | วิเคราะห์: จุดตัดคืออะไร? อะไรคืออุปสรรค? |
| **Decide** | เลือก Ikigai → Mini-experiment → Action Plan |
| **Act** | ลงมือทำ 90 วัน |

---

## 7. 10-10-10 Framework (Suzy Welsh)

### หลักการ
เวลาตัดสินใจ — ถาม 3 คำถาม:

1. ผลลัพธ์ใน **10 นาที** จะเป็นยังไง?
2. ผลลัพธ์ใน **10 เดือน** จะเป็นยังไง?
3. ผลลัพธ์ใน **10 ปี** จะเป็นยังไง?

### วิธีใช้กับ Ikigai
- 10 นาที: การเปลี่ยนเส้นทางชีวิตน่ากลัว
- 10 เดือน: คุณอาจจะยังอยู่ในช่วงทดลอง
- 10 ปี: ถ้าไม่เริ่มตอนนี้ คุณอาจเสียดาย

## สรุป: ใช้ frameworks รวมกัน

| Step | Framework ที่ใช้ |
|------|----------------|
| ทำความเข้าใจตัวเอง | Ikigai 4 วงกลม + Self-reflections |
| ตรวจสอบสมดุลชีวิต | Wheel of Life |
| รู้จุดแข็งของตัวเอง | StrengthsFinder |
| จับประเภทอาชีพที่เหมาะ | Holland Code |
| หาเหตุผลลึก | 5 Whys |
| ตรวจสอบความกลัว | Regret Minimization + 10-10-10 |
| ลงมือทำ | OODA Loop + Action Plan |
