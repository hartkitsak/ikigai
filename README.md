# ikigai

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Ikigai (生き甲斐) is a Japanese philosophy from Okinawa that represents the intersection of what you love, what you're good at, what the world needs, and what you can be paid for. This skill guides Thai speakers through a structured process to discover their ikigai.

## About This Skill

The skill covers five phases:

- **Reflect** — Review past experiences using 40+ guided questions across four dimensions
- **Find** — Map your answers onto the four ikigai circles to locate the intersection
- **Experiment** — Run 14-day mini-experiments to test your findings in the real world
- **Synthesize** — Refine your ikigai statement based on what you learned
- **Act** — Build a 90-day action plan with weekly check-ins to make it real

Each phase builds on the previous one, guiding the user from self-discovery to actionable life direction.

## Usage

Open Claude Code and type `/ikigai` followed by what you need:

```
/ikigai ช่วยหา Ikigai ให้หน่อย
ผมอายุ 28 เป็นโปรแกรมเมอร์ แต่รู้สึกเบื่อ
ไม่รู้ว่าชีวิตควรไปทางไหน
```

```
/ikigai เช็คความคืบหน้า
สัปดาห์ที่แล้วลองสอนเพื่อนใช้ Notion แล้วสนุกดี
อยากปรับ Ikigai Statement
```

```
/ikigai ช่วยถามคำถามสะท้อนตัวเอง
ผมอายุ 35 เป็นผู้จัดการ รู้สึก burnout
อยากรู้ว่าจริงๆ แล้วชอบอะไร
```
