# ikigai — AI Coach หาเส้นทางชีวิต

AI coach ที่ช่วยคนไทยค้นหา **Ikigai** — จุดตัดของสิ่งที่ชอบ สิ่งที่เก่ง สิ่งที่โลกต้องการ และสิ่งที่ทำเงินได้

Skill นี้ใช้กับ [opencode](https://opencode.ai), Claude Code, Cursor หรือ AI agent ที่รองรับ custom skills

## วิธีใช้

```
อ่าน ikigai/SKILL.md แล้วช่วยหา Ikigai ให้หน่อย
```

## โครงสร้าง

```
ikigai/
├── SKILL.md              ← Entry point หลัก
├── workflow/             ← 5 ขั้นตอนค้นหา Ikigai
├── scripts/              ← คำถามสะท้อน, Mini-experiments, Obstacle buster
├── template/             ← Ikigai map, Statement, Action plan
├── example/              ← ตัวอย่างชีวิตจริง 3 กรณี
├── reference/            ← หลักการลึก, Frameworks อื่น, Cheatsheet
└── assets/               ← Venn diagram SVG
```

## สิ่งที่ได้

- ค้นหา Ikigai ผ่าน 4 วงกลม (Love, Good At, World Needs, Paid For)
- 40+ คำถามสะท้อนตัวเอง
- Mini-experiment 14 วันเพื่อพิสูจน์ก่อนลงมือจริง
- Action Plan 90 วัน + Weekly check-in
- รองรับภาษาไทย น้ำเสียงอบอุ่น เป็นกันเอง
