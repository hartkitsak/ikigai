# Mini-Experiments — ทดสอบ Ikigai 14 วัน

---

## วิธีเลือก Experiment

| ถ้า Ikigai ของคุณเกี่ยวกับ | เลือก Experiment |
|--------------------------|-----------------|
| สอน / ให้ความรู้ | #1, #5 |
| สร้างงานสร้างสรรค์ | #2, #7 |
| ดูแล / ช่วยเหลือคน | #3, #8 |
| ขาย / ต่อรอง / การตลาด | #4, #9 |
| ประดิษฐ์ / แก้ปัญหาเทคนิค | #6, #10 |

## 10 Experiments

### #1 The Mini-Class
สอนเรื่องที่คุณสนใจให้เพื่อน 2-3 คน (หรืออัดคลิปลง YouTube สั้นๆ)

| Criteria | คำถาม |
|----------|-------|
| ❤️ สนุกมั้ย? | รู้สึกอยากเตรียมมากกว่าที่จำเป็นมั้ย? |
| ✅ สอนดีมั้ย? | คนฟังเข้าใจมั้ย? เขาบอกว่าสอนเข้าใจง่าย? |
| 🌍 มีคุณค่ามั้ย? | เขาได้อะไรจากคลิปนี้? |
| 💰 ทำเงินได้มั้ย? | มีคนบอกว่าถ้าทำแบบนี้ขายได้มั้ย? |

### #2 The 14-Day Creation Challenge
สร้างอะไรบางอย่างทุกวันเป็นเวลา 14 วัน (รูป, ข้อความ, คลิปสั้น, เพลง, โค้ด)

| Criteria | คำถาม |
|----------|-------|
| ❤️ | เบื่อวันที่เท่าไหร่? หรือรู้สึก "ทำอีกก็ได้"? |
| ✅ | ผลงานวันที่ 14 ดีกว่าวันที่ 1 มั้ย? |
| 🌍 | คนอื่นมีปฏิกิริยายังไง? |
| 💰 | ชิ้นไหนที่ขายได้ / มีคนจ่าย? |

### #3 The Volunteer Week
อาสาช่วยเหลือในด้านที่คุณสนใจ — โดยไม่คิดเงิน

| Criteria | คำถาม |
|----------|-------|
| ❤️ | รู้สึกอยากไปทุกวัน หรือรอให้จบ? |
| ✅ | คุณมีประโยชน์ต่อเค้ามั้ย? |
| 🌍 | เค้าได้อะไรจากคุณ? |
| 💰 | มีช่องทางทำเงินตรงนี้มั้ย? |

### #4 The Side Hustle Tryout
ลงขายสินค้า/บริการเล็กๆ (ของมือสอง, รับจ๊อบเล็กๆ, ขายคลาสออนไลน์)

| Criteria | คำถาม |
|----------|-------|
| ❤️ | สนุกกับกระบวนการขายมั้ย? |
| ✅ | คุณขายเก่งมั้ย? หรือต้องปรับเยอะ? |
| 🌍 | คนซื้อเพราะได้ประโยชน์จริง หรือซื้อเพราะสงสาร? |
| 💰 | มี margin พอทำเป็นรายได้หลักมั้ย? |

### #5 The Mentor Session
ชวนคนที่ทำในสิ่งที่คุณสนใจ ไปกินข้าว/โทรคุย 30 นาที

| Criteria | คำถาม |
|----------|-------|
| ❤️ | ฟังแล้วรู้สึก "อยากทำอันนี้" หรือ "ไม่ใช่เรา"? |
| ✅ | คุณเข้าใจสิ่งที่เค้าพูดมั้ย? หรือมันลึกเกินไป? |
| 🌍 | คุณเห็นปัญหาที่เค้าแก้แล้วคุณก็อยากแก้มั้ย? |
| 💰 | คนสายนี้มีรายได้ที่ "พออยู่ได้" มั้ย? |

### #6 The Fix-It Challenge
หาคนที่มีปัญหา 1 คน แล้วแก้ให้ภายใน 7 วัน

| Criteria | คำถาม |
|----------|-------|
| ❤️ | รู้สึกสนุกกับการแก้ปัญหานี้มั้ย? |
| ✅ | คุณแก้ได้จริงมั้ย? หรือต้องขอความช่วยเหลือ? |
| 🌍 | ปัญหานี้มีคนเป็นเยอะมั้ย? |
| 💰 | คนยอมจ่ายเพื่อแก้ปัญหานี้มั้ย? |

### #7 The Content Pivot
เปลี่ยนช่องทางสื่อที่คุณใช้ ไปเล่าเรื่องที่คุณชอบจริงๆ (เช่น ปกติลงรูปอาหาร ลองลงรูปวาด)

| Criteria | คำถาม |
|----------|-------|
| ❤️ | ทำแล้วสนุกกว่าของเดิมมั้ย? |
| ✅ | content ใหม่มีคุณภาพเท่า/ดีกว่าของเดิมมั้ย? |
| 🌍 | คนมี engagement กับ content ใหม่อย่างไร? |
| 💰 | content ใหม่มีโอกาสทำเงินมั้ย? |

### #8 The Problem Hunt
ถามเพื่อน 10 คน: "ตอนนี้คุณเดือดร้อนเรื่องอะไรที่สุด?"

| Criteria | คำถาม |
|----------|-------|
| ❤️ | คุณสนใจปัญหาของเค้ามั้ย? หรือรู้สึกเฉยๆ? |
| ✅ | คุณช่วยเค้าได้มั้ย? |
| 🌍 | ปัญหานี้เกิดกับคนเยอะมั้ย? |
| 💰 | คนยอมจ่ายเพื่อแก้ปัญหานี้มั้ย? |

### #9 The Pop-Up Shop
ลองขายของ/บริการ 1 อย่าง ในงานอีเวนต์เล็กๆ หรือ online 7 วัน

| Criteria | คำถาม |
|----------|-------|
| ❤️ | สนุกกับการขายมั้ย? |
| ✅ | คุณขายของเก่งมั้ย? |
| 🌍 | คนซื้อเพราะของดี หรือซื้อเพราะเกรงใจ? |
| 💰 | กำไรเท่าไหร่? คุ้มค่าเวลามั้ย? |

### #10 The Deep-Dive
ใช้เวลา 7 วันเรียนทักษะใหม่ที่เกี่ยวข้องกับ Ikigai ที่สงสัย

| Criteria | คำถาม |
|----------|-------|
| ❤️ | เรียนแล้วสนุกมั้ย? หรือฝืน? |
| ✅ | คุณเรียนรู้ได้เร็วแค่ไหน? |
| 🌍 | ทักษะนี้ช่วยแก้ปัญหาอะไรได้บ้าง? |
| 💰 | ทักษะนี้มีตลาดรองรับมั้ย? |

## Scoring

หลัง 14 วันตอบ 4 คำถาม:

| ข้อ | ❤️ | ✅ | 🌍 | 💰 |
|-----|-----|-----|-----|-----|
| Yes | ตื่นเต้น อยากทำต่อ | ทำได้ดี มีคนบอก | มีคนได้ประโยชน์ | คนพร้อมจ่าย |
| No | เบื่อ ไม่อยากทำอีก | ยังไม่ดี ไม่พัฒนาครับ | ไม่เห็น impact | ไม่มีตลาด |
| Maybe | ต้องลองอีกครั้ง | ต้องฝึกอีก | น่ามีแต่ยังไม่แน่ใจ | ยังไม่รู้ |

### ตัดสินใจ

| ผล | ควรทำ |
|-----|-------|
| Yes 3+ ข้อ | **ไปต่อ** — Ikigai ตรงทาง |
| Yes 1-2 ข้อ | **ปรับ** — Ikigai ต้องแก้ หรือ experiment ผิด |
| Yes 0 ข้อ | **กลับไป Step 1** — ต้องค้นหาใหม่ หรือ experiment ไม่ตรง |
